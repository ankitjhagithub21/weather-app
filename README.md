# weather-app
A weather application built using Bootstrap, JavaScript, and OpenWeatherMap API.

## Features
- Displays current weather information for any city
- Shows the temperature, humidity, wind speed, sunrise and sunset time
- Features a responsive design using Bootstrap
- Uses the OpenWeatherMap API to retrieve weather information

## Getting Started
1. Clone the repository:
 (``` git clone https://github.com/ankitjhagithub21/weather-app.git ''') 
2. Go to the project directory:
('''cd weather-app ''')
3. Open index.html in your browser

## Usage
1. Enter the name of a city in the search bar and hit enter.
2. The weather information for that city will be displayed on the page.

## Built With
- Bootstrap
- JavaScript
- OpenWeatherMap API
